# 반도체 FAB 수율 분석 엔지니어 업무 프로세스 및 데이터 온톨로지

## 1. 수율(Yield) 분석 업무 개요

반도체 FAB(Fabrication)에서 수율 분석 엔지니어는 웨이퍼 제조 공정 중 발생하는 수율 저하의 원인을 추적하고, 특이점(Excursion)을 탐지하여 원인 설비(Root Cause Equipment)를 특정하는 역할을 담당한다. 수율 특이점이란 특정 시점 또는 특정 조건에서 수율이 통계적으로 유의미하게 떨어지는 현상을 말한다.

수율 분석의 핵심 목표는 다음과 같다:
- Lot 단위 또는 Wafer 단위의 수율 이상 탐지
- 이상이 발생한 공정 단계(Process Step) 특정
- 해당 공정 단계에서 사용된 설비(Equipment) 및 챔버(Chamber) 특정
- 통계적 검증을 통한 원인 설비 확정
- 개선 조치 후 효과 검증

## 2. 수율 분석 프로세스 (Step-by-Step)

### Step 1: 수율 특이점 탐지 (Excursion Detection)

엔지니어는 매일 수율 모니터링 시스템을 통해 수율 이상을 탐지한다. 주요 방법은 다음과 같다:

SPC(Statistical Process Control) 차트 모니터링 방법은 일별/주별 수율 트렌드를 X-bar 차트와 R 차트로 모니터링하는 것이다. UCL(Upper Control Limit)과 LCL(Lower Control Limit)을 벗어나는 포인트를 탐지한다. 관리 한계는 보통 평균 ± 3시그마로 설정한다.

WAT(Wafer Acceptance Test) 파라미터 모니터링 방법은 WAT 측정값의 분포 변화를 추적하는 것이다. 주요 파라미터로는 Vth(문턱전압, Threshold Voltage), Ion(온전류), Ioff(누설전류), BVdss(항복전압), Rsp(특수저항) 등이 있다.

Bin 분포 분석 방법은 최종 테스트에서 각 Bin(불량 카테고리)의 비율 변화를 추적하는 것이다. 특정 Bin의 비율이 급증하면 해당 불량 모드와 연관된 공정을 우선 조사한다.

### Step 2: 불량 Lot 그룹핑 (Defective Lot Grouping)

수율 특이점이 탐지되면 불량 Lot을 그룹핑한다. 이때 사용하는 테이블은 YIELD_SUMMARY 수율 요약 테이블이다.

그룹핑 기준은 다음과 같다. 동일 제품(Product) 내에서 수율이 평균 - 2시그마 이하인 Lot을 Bad Lot(불량 로트)으로 분류하고, 나머지를 Good Lot(양품 로트)으로 분류한다. 시간 구간(예: 최근 1개월)을 설정하여 분석 범위를 결정한다.

그룹핑 SQL 예시는 다음과 같다. YIELD_SUMMARY 테이블에서 LOT_ID, PRODUCT_ID, YIELD, WAFER_COUNT를 조회하고, YIELD가 AVG_YIELD - 2 * STD_YIELD보다 작으면 BAD로 분류하고 그 외에는 GOOD으로 분류한다. 조건은 PRODUCT_ID가 PROD_A이고 TEST_END_DATE가 2024-01-01부터 2024-01-31 사이인 경우이다.

### Step 3: 공정 이력 추적 (Process History Trace)

Bad Lot과 Good Lot이 각 공정 단계에서 어떤 설비를 사용했는지 추적한다. 이 단계에서 LOT_HISTORY 공정 이력 테이블과 EQUIPMENT_MASTER 설비 마스터 테이블을 조인하여 사용한다.

추적 로직은 다음과 같다. Bad Lot 그룹과 Good Lot 그룹이 특정 공정 단계에서 서로 다른 설비를 사용했는지 비교한다. 만약 Bad Lot이 특정 설비에 집중되어 있다면, 해당 설비가 원인일 가능성이 높다.

### Step 4: 통계적 검증 (Statistical Validation)

설비별 수율 차이가 통계적으로 유의미한지 검증한다. 주로 사용하는 통계 기법은 다음과 같다:

카이제곱 검정(Chi-Square Test)은 Bad/Good Lot의 설비 사용 비율이 독립적인지 검정하는 방법이다. p-value가 0.05보다 작으면 해당 공정단계-설비 조합이 수율과 유의미한 관계가 있다고 판단한다.

t-검정(t-Test)은 특정 설비를 사용한 Lot과 사용하지 않은 Lot 간의 수율 평균 차이를 검정하는 방법이다. 두 그룹의 수율 분포가 정규분포를 따른다고 가정할 때 사용한다.

Kruskal-Wallis 검정은 3개 이상의 설비 그룹 간 수율 차이를 비교할 때 사용하는 비모수 검정 방법이다. 정규분포 가정이 필요 없다.

상관 분석(Correlation Analysis)은 WAT 파라미터와 최종 수율 간의 상관계수를 계산하여 어떤 파라미터가 수율에 가장 큰 영향을 미치는지 파악하는 방법이다. Pearson 상관계수 또는 Spearman 순위상관계수를 사용한다.

### Step 5: 웨이퍼 맵 분석 (Wafer Map Pattern Analysis)

불량 칩의 웨이퍼 상 위치 패턴을 분석한다. WAFER_MAP 웨이퍼 맵 테이블에서 Die 좌표별 Pass/Fail 정보를 추출하여 시각화한다.

주요 웨이퍼 맵 불량 패턴 유형은 다음과 같다. Edge 불량 패턴은 웨이퍼 가장자리에 불량이 집중되는 현상으로, 코팅이나 식각 균일성 문제 또는 Edge Ring 마모가 원인이다. Center 불량 패턴은 웨이퍼 중심부에 불량이 집중되는 현상으로, 가스 흐름 불균일이나 척(Chuck) 문제가 원인이다. Scratch 패턴은 직선형 불량으로 CMP(Chemical Mechanical Polishing, 화학기계적 연마) 설비 문제가 원인이다. Random 불량 패턴은 무작위로 분포하는 불량으로 파티클(Particle) 오염이 원인이다. Cluster 불량 패턴은 특정 영역에 불량이 집중되는 현상으로 리소그래피 렌즈 오염이나 마스크 결함이 원인이다. Ring 패턴은 원형 불량으로 스핀코팅 속도 이상이나 온도 분포 불균일이 원인이다.

### Step 6: 원인 설비 확정 및 조치 (Root Cause Confirmation)

통계 분석과 웨이퍼 맵 분석 결과를 종합하여 원인 설비를 확정한다. 확정 후에는 설비 PM(Preventive Maintenance, 예방정비)을 실시하고, PM 전후 수율을 비교하여 효과를 검증한다.

## 3. 핵심 데이터 테이블 정의

### 3.1 YIELD_SUMMARY 수율 요약 테이블

YIELD_SUMMARY 수율 요약 테이블은 Lot 단위의 최종 수율 정보를 저장하며, 수율 분석의 출발점이 되는 가장 중요한 테이블이다. 이 테이블의 컬럼 구성은 다음과 같다.

LOT_ID 컬럼은 VARCHAR(20) 타입의 Lot 고유 식별자로, 예를 들어 L240101001과 같은 형식이다. PRODUCT_ID 컬럼은 VARCHAR(15) 타입의 제품 코드로, PROD_A나 PROD_B 같은 제품을 식별한다. PROCESS_FLOW 컬럼은 VARCHAR(20) 타입의 공정 플로우 코드이다. LOT_START_DATE 컬럼은 DATETIME 타입의 Lot 투입 일시이다. TEST_END_DATE 컬럼은 DATETIME 타입의 최종 테스트 완료 일시이다. WAFER_COUNT 컬럼은 INT 타입의 Lot 내 웨이퍼 수로 보통 25매이다. TOTAL_DIE 컬럼은 INT 타입의 총 다이 수이다. GOOD_DIE 컬럼은 INT 타입의 양품 다이 수이다. YIELD 컬럼은 DECIMAL(5,2) 타입의 수율 퍼센트로 GOOD_DIE를 TOTAL_DIE로 나눈 뒤 100을 곱한 값이다. BIN1_COUNT 컬럼은 INT 타입의 Bin1 양품 수량이다. BIN2_COUNT 컬럼은 INT 타입의 Bin2 속도불량 수량이다. BIN3_COUNT 컬럼은 INT 타입의 Bin3 누설전류불량 수량이다. BIN4_COUNT 컬럼은 INT 타입의 Bin4 기능불량 수량이다. BIN5_COUNT 컬럼은 INT 타입의 Bin5 기타불량 수량이다.

### 3.2 LOT_HISTORY 공정 이력 테이블

LOT_HISTORY 공정 이력 테이블은 Lot이 각 공정 단계에서 어떤 설비의 어떤 챔버를 사용했는지와 언제 처리되었는지를 기록하는 테이블이며, 원인 설비 추적의 핵심 테이블이다. 이 테이블의 컬럼 구성은 다음과 같다.

LOT_ID 컬럼은 VARCHAR(20) 타입의 Lot 고유 식별자이다. STEP_SEQ 컬럼은 INT 타입의 공정 순서 번호이다. STEP_ID 컬럼은 VARCHAR(30) 타입의 공정 단계 코드로, PHOTO_L1이나 ETCH_L1이나 DEP_L2 같은 형식이다. STEP_NAME 컬럼은 VARCHAR(100) 타입의 공정 단계명으로, 1st Metal Photo나 Gate Etch 같은 이름이다. EQUIPMENT_ID 컬럼은 VARCHAR(20) 타입의 설비 고유 식별자로, PHOTO-01이나 ETCH-03 같은 형식이다. CHAMBER_ID 컬럼은 VARCHAR(10) 타입의 챔버 식별자로, CH_A나 CH_B 같은 형식이다. RECIPE_ID 컬럼은 VARCHAR(30) 타입의 레시피(공정 조건) 코드이다. PROCESS_START 컬럼은 DATETIME 타입의 공정 시작 일시이다. PROCESS_END 컬럼은 DATETIME 타입의 공정 종료 일시이다. OPERATOR_ID 컬럼은 VARCHAR(15) 타입의 작업자 ID이다. LOT_STATUS 컬럼은 VARCHAR(10) 타입의 처리 상태로, COMPLETED나 ABORTED나 HOLD 중 하나이다.

### 3.3 EQUIPMENT_MASTER 설비 마스터 테이블

EQUIPMENT_MASTER 설비 마스터 테이블은 FAB 내 모든 설비의 기본 정보를 관리하는 테이블이다. 이 테이블의 컬럼 구성은 다음과 같다.

EQUIPMENT_ID 컬럼은 VARCHAR(20) 타입의 설비 고유 식별자이다. EQUIPMENT_NAME 컬럼은 VARCHAR(50) 타입의 설비명으로, ASML NXT:1980Di 같은 이름이다. EQUIPMENT_TYPE 컬럼은 VARCHAR(20) 타입의 설비 유형으로, PHOTO(노광)나 ETCH(식각)나 CVD(화학기상증착)나 PVD(물리기상증착)나 CMP(화학기계적연마)나 DIFF(확산)나 IMP(이온주입) 같은 유형이다. VENDOR 컬럼은 VARCHAR(30) 타입의 제조사로, ASML이나 LAM Research나 Applied Materials(AMAT)나 Tokyo Electron(TEL) 같은 회사이다. INSTALL_DATE 컬럼은 DATE 타입의 설치일이다. CHAMBER_COUNT 컬럼은 INT 타입의 챔버 수이다. CURRENT_STATUS 컬럼은 VARCHAR(15) 타입의 현재 상태로, RUN(가동 중)이나 DOWN(고장)이나 PM(정비 중)이나 IDLE(대기 중) 중 하나이다. AREA 컬럼은 VARCHAR(10) 타입의 설치 구역으로, BAY1이나 BAY2 같은 구역이다.

### 3.4 WAT_DATA 웨이퍼 수락 테스트 데이터 테이블

WAT_DATA 웨이퍼 수락 테스트(Wafer Acceptance Test) 데이터 테이블은 공정 중간 단계에서의 전기적 특성 측정 결과를 저장하는 테이블이다. 이 테이블의 컬럼 구성은 다음과 같다.

LOT_ID 컬럼은 VARCHAR(20) 타입의 Lot 고유 식별자이다. WAFER_ID 컬럼은 VARCHAR(5) 타입의 웨이퍼 번호로 01부터 25까지이다. SITE_ID 컬럼은 INT 타입의 측정 사이트 번호로 웨이퍼 내 측정 위치를 나타낸다. PARAM_NAME 컬럼은 VARCHAR(30) 타입의 파라미터명으로, VTH_NMOS(NMOS 문턱전압)나 VTH_PMOS(PMOS 문턱전압)나 ION_N(NMOS 온전류)나 IOFF_N(NMOS 누설전류)나 BVDSS(항복전압)나 RSP(특수저항) 같은 파라미터이다. PARAM_VALUE 컬럼은 DECIMAL(15,6) 타입의 측정값이다. SPEC_LOW 컬럼은 DECIMAL(15,6) 타입의 규격 하한이다. SPEC_HIGH 컬럼은 DECIMAL(15,6) 타입의 규격 상한이다. MEASURE_DATE 컬럼은 DATETIME 타입의 측정 일시이다. JUDGE 컬럼은 VARCHAR(5) 타입의 판정 결과로 PASS 또는 FAIL이다.

### 3.5 WAFER_MAP 웨이퍼 맵 테이블

WAFER_MAP 웨이퍼 맵 테이블은 최종 테스트 후 각 Die의 좌표별 Pass/Fail 결과를 저장하는 테이블이다. 이 테이블의 컬럼 구성은 다음과 같다.

LOT_ID 컬럼은 VARCHAR(20) 타입의 Lot 고유 식별자이다. WAFER_ID 컬럼은 VARCHAR(5) 타입의 웨이퍼 번호이다. DIE_X 컬럼은 INT 타입의 Die X좌표이다. DIE_Y 컬럼은 INT 타입의 Die Y좌표이다. BIN_CODE 컬럼은 INT 타입의 Bin 코드로 1은 양품이고 2부터 9는 각 불량 유형이다. HARD_BIN 컬럼은 INT 타입의 하드 빈 코드로 하드웨어 수준에서의 분류이다. SOFT_BIN 컬럼은 INT 타입의 소프트 빈 코드로 소프트웨어 수준에서의 상세 분류이다. TEST_DATE 컬럼은 DATETIME 타입의 테스트 일시이다.

### 3.6 DEFECT_DATA 결함 검사 데이터 테이블

DEFECT_DATA 결함 검사 데이터 테이블은 인라인 검사 장비로 측정한 결함 정보를 저장하는 테이블이다. 검사 장비로는 KLA Instruments사의 결함 검사 장비나 Applied Materials사의 검사 장비가 사용된다. 이 테이블의 컬럼 구성은 다음과 같다.

LOT_ID 컬럼은 VARCHAR(20) 타입의 Lot 고유 식별자이다. WAFER_ID 컬럼은 VARCHAR(5) 타입의 웨이퍼 번호이다. INSPECTION_STEP 컬럼은 VARCHAR(30) 타입의 검사 공정 단계이다. DEFECT_ID 컬럼은 INT 타입의 결함 고유 번호이다. DEFECT_X 컬럼은 DECIMAL(10,3) 타입의 결함 X좌표로 밀리미터 단위이다. DEFECT_Y 컬럼은 DECIMAL(10,3) 타입의 결함 Y좌표로 밀리미터 단위이다. DEFECT_SIZE 컬럼은 DECIMAL(8,3) 타입의 결함 크기로 마이크로미터 단위이다. DEFECT_CLASS 컬럼은 VARCHAR(20) 타입의 결함 분류로, PARTICLE(파티클)이나 SCRATCH(스크래치)나 PATTERN(패턴 결함)이나 RESIDUE(잔류물)나 VOID(공극) 같은 유형이다. DEFECT_COUNT 컬럼은 INT 타입의 웨이퍼 당 총 결함 수이다. INSPECT_DATE 컬럼은 DATETIME 타입의 검사 일시이다.

### 3.7 EQP_SENSOR_DATA 설비 센서 데이터 테이블

EQP_SENSOR_DATA 설비 센서 데이터 테이블은 설비의 실시간 센서 데이터를 저장하는 FDC(Fault Detection and Classification, 설비 이상 탐지 및 분류) 데이터 테이블이다. 이 테이블의 컬럼 구성은 다음과 같다.

EQUIPMENT_ID 컬럼은 VARCHAR(20) 타입의 설비 고유 식별자이다. CHAMBER_ID 컬럼은 VARCHAR(10) 타입의 챔버 식별자이다. LOT_ID 컬럼은 VARCHAR(20) 타입의 처리 중인 Lot ID이다. WAFER_ID 컬럼은 VARCHAR(5) 타입의 처리 중인 Wafer ID이다. SENSOR_NAME 컬럼은 VARCHAR(30) 타입의 센서명으로, TEMPERATURE(온도 센서)나 PRESSURE(압력 센서)나 RF_POWER(RF 파워 센서)나 GAS_FLOW(가스 유량 센서)나 BIAS_VOLTAGE(바이어스 전압 센서) 같은 센서이다. SENSOR_VALUE 컬럼은 DECIMAL(15,6) 타입의 센서 측정값이다. SENSOR_TIME 컬럼은 DATETIME 타입의 측정 시각이다. RECIPE_STEP 컬럼은 INT 타입의 레시피 스텝 번호이다. UCL 컬럼은 DECIMAL(15,6) 타입의 관리 상한(Upper Control Limit)이다. LCL 컬럼은 DECIMAL(15,6) 타입의 관리 하한(Lower Control Limit)이다.

### 3.8 PM_HISTORY 설비 예방정비 이력 테이블

PM_HISTORY 설비 예방정비(Preventive Maintenance) 이력 테이블은 설비의 정비 이력을 기록하는 테이블이다. 이 테이블의 컬럼 구성은 다음과 같다.

PM_ID 컬럼은 VARCHAR(20) 타입의 PM 고유 식별자이다. EQUIPMENT_ID 컬럼은 VARCHAR(20) 타입의 설비 식별자이다. CHAMBER_ID 컬럼은 VARCHAR(10) 타입의 챔버 식별자이다. PM_TYPE 컬럼은 VARCHAR(20) 타입의 PM 유형으로, DAILY(일일 점검)나 WEEKLY(주간 점검)나 MONTHLY(월간 점검)나 QUARTERLY(분기 점검)나 ANNUAL(연간 점검) 같은 유형이다. PM_START 컬럼은 DATETIME 타입의 PM 시작 일시이다. PM_END 컬럼은 DATETIME 타입의 PM 종료 일시이다. PM_DESCRIPTION 컬럼은 VARCHAR(200) 타입의 PM 내용이다. PARTS_REPLACED 컬럼은 VARCHAR(200) 타입의 교체 부품 목록이다. TECHNICIAN_ID 컬럼은 VARCHAR(15) 타입의 작업자 ID이다.

## 4. 수율 특이점 분석 로직 상세

### 4.1 설비 기여도 분석 (Equipment Contribution Analysis)

설비 기여도 분석은 각 공정 단계에서 어떤 설비가 수율 저하에 기여하는지를 정량화하는 방법이다.

분석 절차는 다음과 같다. 첫째, YIELD_SUMMARY 수율 요약 테이블에서 분석 대상 Lot을 선정한다. 둘째, 선정된 Lot의 LOT_HISTORY 공정 이력을 조회하여 각 공정 단계별 사용 설비를 매핑한다. 셋째, 각 공정 단계(STEP_ID) 곱하기 설비(EQUIPMENT_ID) 조합별로 평균 수율을 계산한다. 넷째, 카이제곱 검정(Chi-Square Test)으로 각 공정 단계별 설비 간 수율 차이의 유의성을 검증한다. 다섯째, p-value가 낮은 순으로 공정 단계를 정렬하여 의심 공정을 선별한다.

핵심 SQL은 LOT_HISTORY 테이블과 YIELD_SUMMARY 테이블을 LOT_ID로 조인하고, STEP_ID와 EQUIPMENT_ID별로 그룹핑하여 LOT_COUNT(Lot 수), AVG_YIELD(평균 수율), STD_YIELD(수율 표준편차), MIN_YIELD(최소 수율), MAX_YIELD(최대 수율)를 계산하는 쿼리이다.

### 4.2 Commonality Analysis 공통점 분석

Commonality Analysis 공통점 분석은 Bad Lot 그룹이 공통적으로 거쳐간 설비 또는 챔버를 찾는 분석 방법이다.

Commonality Rate 공통점 비율 계산 방법은 다음과 같다. Bad Lot Commonality 비율은 해당 설비를 사용한 Bad Lot 수를 전체 Bad Lot 수로 나누고 100을 곱한 값이다. Good Lot Commonality 비율은 해당 설비를 사용한 Good Lot 수를 전체 Good Lot 수로 나누고 100을 곱한 값이다. Delta 차이값은 Bad Lot Commonality에서 Good Lot Commonality를 뺀 값이다.

Delta가 클수록(양수), 해당 설비가 수율 저하의 원인일 가능성이 높다. 일반적으로 Delta가 30% 이상이면 강한 의심 대상이다.

### 4.3 Multi-Step Correlation 다단계 상관 분석

Multi-Step Correlation 다단계 상관 분석은 한 공정 단계의 설비 효과만으로는 원인을 특정할 수 없는 경우에 여러 공정 단계의 설비 조합 효과를 분석하는 방법이다.

예를 들어, Photo(노광) 공정의 설비 A와 Etch(식각) 공정의 설비 B를 연속으로 사용한 Lot의 수율이 특히 낮다면, 두 설비 간의 상호작용 효과(Interaction Effect)가 존재한다고 판단한다. 이는 이원분산분석(Two-way ANOVA)으로 검증할 수 있다.

### 4.4 시계열 분석 (Time Series Analysis)

시계열 분석(Time Series Analysis)은 설비의 성능이 시간에 따라 열화되는 패턴을 탐지하는 방법이다.

PM 주기 영향 분석은 PM_HISTORY 설비 예방정비 이력 테이블과 YIELD_SUMMARY 수율 요약 테이블을 조인하여 수행한다. 설비의 마지막 PM 이후 경과 시간과 수율 간의 관계를 분석한다. PM 직후에는 수율이 높다가 PM 주기가 다가올수록 수율이 떨어지는 톱니파 패턴(Sawtooth Pattern)이 있으면 PM 주기를 단축해야 한다.

돌발 이상 탐지에는 CUSUM(Cumulative Sum, 누적합 관리도) 차트 또는 EWMA(Exponentially Weighted Moving Average, 지수가중이동평균 관리도) 차트를 사용하여 수율의 점진적 변화를 탐지한다.

## 5. 주요 공정 단계와 관련 설비 유형

### 5.1 리소그래피 공정 (Photo/Lithography Process)

리소그래피 공정(Photo/Lithography)은 반도체 패턴을 형성하는 핵심 공정이다. 사용 설비로는 스캐너(Scanner)와 트랙(Track/Coater-Developer)이 있다. 주요 파라미터로는 CD(Critical Dimension, 패턴의 핵심 치수로 나노미터 단위), Overlay(층간 정렬 정밀도), Focus(초점), Dose(노광량)가 있다. 대표 설비로는 ASML사의 NXT:1980Di 스캐너, ASML사의 TWINSCAN 스캐너, Tokyo Electron사의 CLEAN TRACK ACT12 트랙 설비가 있다. 수율에 미치는 영향으로는, Overlay 불량이 발생하면 층간 정렬 불량으로 이어져 전기적 단락이나 단선이 발생한다.

### 5.2 식각 공정 (Etch Process)

식각 공정(Etch)은 패턴 형성을 위해 불필요한 막을 제거하는 공정이다. 사용 설비로는 건식 식각기(Dry Etcher)와 습식 식각조(Wet Bench)가 있다. 주요 파라미터로는 Etch Rate(식각 속도), Selectivity(선택비), Profile Angle(식각 프로파일 각도), Uniformity(균일성)가 있다. 대표 설비로는 LAM Research사의 2300 Kiyo 식각기, Tokyo Electron사의 Tactras 식각기, Applied Materials사의 Centura 식각기가 있다. 수율에 미치는 영향으로는, Under-etch(부족 식각)이면 잔류물이 남아 단락이 발생하고, Over-etch(과도 식각)이면 하부층이 손상된다.

### 5.3 증착 공정 (Deposition Process)

증착 공정은 반도체 위에 얇은 막을 형성하는 공정으로, 세 가지 주요 방식이 있다. CVD(Chemical Vapor Deposition, 화학기상증착) 공정은 화학 반응을 이용하여 박막을 증착하는 방법으로, 박막 두께와 균일성과 응력이 주요 파라미터이다. PVD(Physical Vapor Deposition, 물리기상증착) 공정은 스퍼터링 방식으로 금속 배선을 형성하는 방법이다. ALD(Atomic Layer Deposition, 원자층증착) 공정은 원자층 단위로 정밀하게 증착하는 방법으로, High-k 절연막 형성에 사용된다. 대표 설비로는 Applied Materials사의 Producer 증착기, Applied Materials사의 Endura PVD 시스템, ASM International사의 Epsilon 증착기, Jusung Engineering사의 증착 설비가 있다. 수율에 미치는 영향으로는, 두께 불균일이 발생하면 전기적 특성 변동으로 이어져 WAT 파라미터 이상이 발생한다.

### 5.4 CMP 화학기계적 연마 공정 (Chemical Mechanical Polishing Process)

CMP(Chemical Mechanical Polishing, 화학기계적 연마) 공정은 웨이퍼 표면을 평탄화하는 공정이다. 사용 설비로는 CMP 폴리셔(Polisher)가 있다. 주요 파라미터로는 Removal Rate(연마 속도), Uniformity(균일성), Dishing(디싱 현상), Erosion(침식 현상)이 있다. 대표 설비로는 Applied Materials사의 Reflexion CMP 시스템과 Ebara사의 F-REX CMP 시스템이 있다. 수율에 미치는 영향으로는, 과연마가 발생하면 금속 배선 두께가 감소하여 저항이 증가하고, 미연마가 발생하면 잔류물로 인해 단락이 발생한다.

### 5.5 이온주입 공정 (Ion Implantation Process)

이온주입 공정(Ion Implantation)은 반도체에 불순물을 주입하여 전기적 특성을 조절하는 공정이다. 사용 설비로는 이온주입기(Ion Implanter)가 있다. 주요 파라미터로는 Dose(주입량), Energy(에너지), Tilt Angle(경사 각도), Uniformity(균일성)가 있다. 대표 설비로는 Applied Materials사의 VIISta 이온주입기와 Axcelis Technologies사의 Purion 이온주입기가 있다. 수율에 미치는 영향으로는, Dose 불균일이 발생하면 Vth(문턱전압) 변동으로 이어져 소자 특성 이상이 발생한다.

### 5.6 열처리 공정 (Diffusion/Furnace/Thermal Process)

열처리 공정(Diffusion/Furnace)은 고온에서 산화막 형성이나 확산 처리를 수행하는 공정이다. 사용 설비로는 수직형 퍼니스(Vertical Furnace)와 RTP(Rapid Thermal Processing, 급속 열처리) 설비가 있다. 주요 파라미터로는 Temperature(온도), Time(시간), Gas Flow(가스 유량), Uniformity(균일성)가 있다. 대표 설비로는 Tokyo Electron사의 Alpha 퍼니스, ASM International사의 A412 퍼니스, Applied Materials사의 Vantage RTP 설비가 있다. 수율에 미치는 영향으로는, 온도 불균일이 발생하면 산화막 두께 변동으로 이어져 Gate 특성 이상이 발생한다.

### 5.7 세정 공정 (Cleaning Process)

세정 공정(Cleaning)은 웨이퍼 표면의 오염물을 제거하는 공정이다. 사용 설비로는 습식 세정조(Wet Bench)와 단매 세정기(Single Wafer Cleaner)가 있다. 주요 파라미터로는 화학액 농도, 온도, 시간, 파티클 제거율(Particle Removal Rate)이 있다. 대표 설비로는 DNS(Dainippon Screen)사의 SU-3200 습식 세정기, Screen사의 SPM 세정기, SEMES사의 OMEGA 세정기가 있다. 수율에 미치는 영향으로는, 세정 불량이 발생하면 잔류 파티클이나 오염물이 남아 후속 공정에서 결함이 발생한다.

## 6. 실전 분석 시나리오 예시

### 시나리오 1: ETCH 설비 챔버 문제로 인한 수율 저하

현상: PROD_A 제품의 수율이 1월 3주차부터 3%p 하락하였다.

분석 과정은 다음과 같다. 첫째, YIELD_SUMMARY 수율 요약 테이블에서 1월 전체 Lot을 조회하여 Bad Lot(수율이 평균-2시그마 미만)을 분류한 결과 15개 Bad Lot을 선별하였다. 둘째, LOT_HISTORY 공정 이력 테이블에서 Bad Lot 15개와 Good Lot 85개의 공정 이력을 비교하였다. 셋째, ETCH_L3(3차 금속 식각) 단계에서 Commonality 공통점 분석을 수행한 결과, ETCH-03 설비의 CH_B 챔버에서 Bad Lot Commonality가 93%, Good Lot Commonality가 22%, Delta가 71%로 나타났으며, 카이제곱 검정(Chi-Square Test) p-value가 0.00001로 매우 유의하였다. 넷째, EQP_SENSOR_DATA 설비 센서 데이터 테이블에서 ETCH-03 설비 CH_B 챔버의 RF_POWER 센서 트렌드를 확인한 결과, 1월 3주차부터 RF_POWER가 점진적으로 하락하는 추세를 발견하였다. 다섯째, WAFER_MAP 웨이퍼 맵 분석에서 Bad Lot의 웨이퍼에서 Edge 불량 패턴이 공통적으로 관찰되었다. 결론은, ETCH-03 설비 CH_B 챔버의 Edge Ring 마모로 인한 RF 전달 효율 저하가 수율 저하의 원인이었다. 조치로 Edge Ring 교체 PM(예방정비)을 실시한 후 수율 회복을 확인하였다.

### 시나리오 2: Photo-Etch 상호작용으로 인한 간헐적 수율 저하

현상: PROD_B 제품에서 간헐적으로 수율이 급락하는 Lot이 발생한다. 단일 공정 분석으로는 원인이 특정되지 않는다.

분석 과정은 다음과 같다. 첫째, 단일 공정 단계별 설비 기여도 분석에서는 p-value가 0.1보다 커서 유의한 설비가 없었다. 둘째, Multi-Step Correlation 다단계 상관 분석을 실시하여 PHOTO_L2 공정과 ETCH_L2 공정을 동시에 고려하였다. 셋째, PHOTO-02 설비와 ETCH-01 설비 조합에서만 수율이 현저히 낮았다(평균 82.3% 대 전체 평균 91.5%). 넷째, Two-way ANOVA 이원분산분석 결과 교호작용(Interaction) 항의 p-value가 0.003으로 유의하였다. 다섯째, WAT_DATA 웨이퍼 수락 테스트 데이터에서 해당 조합의 CD(Critical Dimension, 핵심 치수)가 규격 하한에 근접함을 확인하였다. 결론은, PHOTO-02 설비의 Dose 설정이 약간 낮은 상태에서 ETCH-01 설비의 Etch Rate가 다른 설비 대비 높아 두 효과가 합쳐져 CD가 규격 하한을 벗어난 것이 원인이었다. 조치로 PHOTO-02 설비의 Dose를 2% 상향 조정하고, ETCH-01 설비의 레시피를 미세 조정하였다.

### 시나리오 3: PM 주기 관련 수율 열화

현상: DIFF-02(확산로) 설비를 사용한 Lot의 수율이 점진적으로 하락하다가 PM(예방정비) 후 회복되는 패턴이 반복된다.

분석 과정은 다음과 같다. 첫째, PM_HISTORY 설비 예방정비 이력 테이블에서 DIFF-02 설비의 PM 이력을 조회하였다. 둘째, LOT_HISTORY 공정 이력 테이블에서 DIFF-02 설비를 사용한 Lot을 추출하고, 각 Lot의 처리 시점이 마지막 PM으로부터 며칠 경과했는지 계산하였다. 셋째, PM 경과일수와 수율 간 Pearson 상관계수가 -0.72로 강한 음의 상관관계가 확인되었다. 넷째, PM 후 20일 경과 시점부터 수율이 유의미하게 하락하기 시작하였다. 다섯째, EQP_SENSOR_DATA 설비 센서 데이터에서 TEMPERATURE(온도) 센서의 Uniformity(균일성)가 PM 후 시간이 지남에 따라 악화되는 것을 확인하였다. 결론은, DIFF-02 설비의 히터 열화로 인한 온도 균일성 저하가 원인이며 현재 Monthly(월간) PM 주기로는 부족하다는 것이었다. 조치로 PM 주기를 Monthly에서 Bi-weekly(격주)로 단축하고, 히터 모듈 교체 일정을 수립하였다.

## 7. 데이터 간 관계(Join Key) 정리

테이블 간의 조인 관계는 다음과 같다. YIELD_SUMMARY 수율 요약 테이블과 LOT_HISTORY 공정 이력 테이블은 LOT_ID 컬럼으로 조인한다. LOT_HISTORY 공정 이력 테이블과 EQUIPMENT_MASTER 설비 마스터 테이블은 EQUIPMENT_ID 컬럼으로 조인한다. YIELD_SUMMARY 수율 요약 테이블과 WAT_DATA 웨이퍼 수락 테스트 데이터 테이블은 LOT_ID 컬럼으로 조인한다. YIELD_SUMMARY 수율 요약 테이블과 WAFER_MAP 웨이퍼 맵 테이블은 LOT_ID 컬럼과 WAFER_ID 컬럼을 함께 사용하여 조인한다. LOT_HISTORY 공정 이력 테이블과 DEFECT_DATA 결함 검사 데이터 테이블은 LOT_ID 컬럼과 WAFER_ID 컬럼으로 조인하며, INSPECTION_STEP 컬럼이 STEP_ID 컬럼에 대응한다. LOT_HISTORY 공정 이력 테이블과 EQP_SENSOR_DATA 설비 센서 데이터 테이블은 EQUIPMENT_ID 컬럼과 LOT_ID 컬럼과 WAFER_ID 컬럼으로 조인한다. EQUIPMENT_MASTER 설비 마스터 테이블과 PM_HISTORY 설비 예방정비 이력 테이블은 EQUIPMENT_ID 컬럼으로 조인한다. EQP_SENSOR_DATA 설비 센서 데이터 테이블과 EQUIPMENT_MASTER 설비 마스터 테이블은 EQUIPMENT_ID 컬럼으로 조인한다.

## 8. 분석 도구 및 환경

엔지니어들이 사용하는 주요 분석 도구는 다음과 같다.

SPC(Statistical Process Control, 통계적 공정 관리) 시스템은 실시간 수율 모니터링 및 알람 기능을 제공하며, InfinityQS사의 SPC 솔루션이나 Camstar사의 MES/SPC 시스템이 대표적이다.

FDC(Fault Detection and Classification, 설비 이상 탐지 및 분류) 시스템은 설비 센서 데이터를 실시간으로 감시하며, KLA Instruments사의 Maestro FDC 시스템이나 PDF Solutions사의 Exensio FDC 플랫폼이 대표적이다.

통계 소프트웨어로는 SAS Institute사의 JMP 통계 분석 소프트웨어, Minitab LLC사의 Minitab 통계 소프트웨어, Python 프로그래밍 언어의 SciPy와 pandas 라이브러리를 사용하여 통계 검정을 수행한다.

데이터베이스로는 Oracle사의 Oracle Database나 TmaxData사의 Tibero Database에서 SQL 쿼리를 사용하여 데이터를 추출한다.

시각화 도구로는 TIBCO사의 Spotfire 분석 플랫폼, Salesforce사의 Tableau 시각화 도구, Python matplotlib 시각화 라이브러리를 사용하여 웨이퍼 맵 및 트렌드 차트를 시각화한다.

자동화 플랫폼으로는 수율 관리 시스템(Yield Management System, YMS)이 위의 분석 로직을 자동 실행하고 결과를 리포트로 생성한다.

## 9. 수율 분석 용어 사전

수율(Yield)은 전체 다이 중 양품 비율을 의미한다. 특이점(Excursion)은 통계적으로 유의미한 수율 이탈 현상을 말한다. Bin 코드는 테스트 결과에 따른 불량 분류 코드이다. WAT(Wafer Acceptance Test, 웨이퍼 수락 테스트)는 웨이퍼 수준의 전기적 특성을 측정하는 테스트이다. SPC(Statistical Process Control, 통계적 공정 관리)는 공정의 안정성을 통계적으로 관리하는 방법이다. FDC(Fault Detection and Classification, 설비 이상 탐지 및 분류)는 설비의 이상 상태를 탐지하고 분류하는 시스템이다. PM(Preventive Maintenance, 예방정비)은 설비의 고장을 예방하기 위해 정기적으로 수행하는 정비 활동이다. CD(Critical Dimension, 핵심 치수)는 반도체 패턴의 핵심 치수로 나노미터 단위로 측정된다. Overlay는 반도체 층간 정렬 정밀도를 의미한다. Vth(Threshold Voltage, 문턱전압)는 MOSFET 트랜지스터가 동작하기 시작하는 전압이다. CMP(Chemical Mechanical Polishing, 화학기계적 연마)는 웨이퍼 표면을 화학적 및 기계적으로 연마하여 평탄화하는 공정이다. ALD(Atomic Layer Deposition, 원자층증착)는 원자층 단위로 정밀하게 박막을 증착하는 공정이다. CVD(Chemical Vapor Deposition, 화학기상증착)는 화학 반응을 이용하여 박막을 증착하는 공정이다. PVD(Physical Vapor Deposition, 물리기상증착)는 물리적 방법으로 금속 박막을 증착하는 공정이다. RTP(Rapid Thermal Processing, 급속 열처리)는 반도체 웨이퍼를 급속으로 가열하여 처리하는 열처리 공정이다. Commonality(공통점 비율)는 Bad Lot이 공통으로 거쳐간 설비의 비율을 나타내는 지표이다. CUSUM(Cumulative Sum, 누적합 관리도)은 공정 변화를 누적합으로 탐지하는 관리도 기법이다. EWMA(Exponentially Weighted Moving Average, 지수가중이동평균 관리도)는 최근 데이터에 더 큰 가중치를 부여하여 공정 변화를 탐지하는 관리도 기법이다.
